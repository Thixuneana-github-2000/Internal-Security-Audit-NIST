# Internal-Security-Audit-NIST
Botium Toys (Case Study) related Internal security Audit

Project Overview 

This project demonstrates an internal IT security audit conducted using the NIST Cybersecurity Framework (CSF). The audit evaluates security controls, compliance best practices, and risk exposure for a fictional company expanding into global online markets. 

Scenario 

Botium Toys is a small U.S.-based business expanding internationally. The IT department conducted an internal audit to: 

Identify security risks 

Assess compliance posture 

Evaluate existing controls 

Recommend improvements 

 

Framework Used 

NIST Cybersecurity Framework (CSF) 

Risk Assessment Methodology 

Controls & Compliance Checklist 

Security Gap Analysis 

 

Key Activities Performed 

 Reviewed audit scope and goals 
 Analyzed risk assessment report 
 Evaluated security controls 
 Assessed compliance best practices 
 Identified security gaps 
 Provided recommendations 

Identified Risk Areas 

Inadequate compliance documentation 

Weak internal control processes 

Potential data protection gaps (EU operations) 

Payment processing compliance exposure
